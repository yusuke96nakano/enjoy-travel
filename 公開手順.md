# Travel AI 公開手順

## 目的

同じWi-Fiでなくても、スマホからTravel AIを開けるようにする。

## 先に確認すること

- GitHubアカウント
- Vercelアカウント
- Macのコマンドラインデベロッパーツール

`git`のインストール確認が出た場合は、`インストール`を押してください。

## 1. GitHubにアップロード

ターミナルで以下を実行します。

```bash
cd "/Users/nakanoyuusuke/Documents/Codex/2026-06-25/codex-web-mvp-1-web-1"
git init
git add .
git commit -m "Initial Travel AI"
git branch -M main
git remote add origin https://github.com/ユーザー名/travel-ai.git
git push -u origin main
```

`ユーザー名`はGitHubのユーザー名に置き換えます。

## 2. Vercelで公開

1. Vercelを開く
2. `Add New...` → `Project`
3. GitHubの`travel-ai`を選択
4. Framework Presetが`Next.js`になっていることを確認
5. Deployを押す

## 3. スマホで開く

Vercelの公開URLをスマホのSafariで開きます。

## 注意

現在の保存データはブラウザー内保存です。

そのため、公開URLでスマホから使えるようになっても、Macとスマホのデータは自動同期されません。
当面は設定画面のバックアップ機能でデータを移動します。

将来的にはSupabaseなどのクラウド保存に切り替えると、Macとスマホで同じデータを見られるようになります。
